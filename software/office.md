# Office

## 快捷键

### 通用快捷键

| 功能             | 快捷键                    |
| ---------------- | ------------------------- |
| 新建文档         | `Ctrl + N`                |
| 打开文档         | `Ctrl + O`                |
| 保存文档         | `Ctrl + S`                |
| 另存为           | `F12`                     |
| 打印             | `Ctrl + P`                |
| 撤销             | `Ctrl + Z`                |
| 重做             | `Ctrl + Y` 或 `F4`        |
| 剪切             | `Ctrl + X`                |
| 复制             | `Ctrl + C`                |
| 粘贴             | `Ctrl + V`                |
| 粘贴为纯文本     | `Ctrl + Alt + V`          |
| 全选             | `Ctrl + A`                |
| 查找             | `Ctrl + F`                |
| 替换             | `Ctrl + H`                |
| 关闭当前窗口     | `Ctrl + W` 或 `Ctrl + F4` |
| 退出应用程序     | `Alt + F4`                |
| 切换窗口         | `Alt + Tab`               |
| 打开帮助         | `F1`                      |
| 重复上一步操作   | `F4`                      |
| 打开“选项”对话框 | `Alt + F, T`              |
